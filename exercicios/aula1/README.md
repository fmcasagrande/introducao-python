# Exercícios — Aula 1

Resolva cada exercício num arquivo `.py` próprio (ou notebook, se preferir). Tente sem consultar a aula; se travar, volte aos exemplos.

## 1. Conversão de temperatura

Peça ao usuário uma temperatura em Celsius (use `input()`) e imprima o valor equivalente em Fahrenheit. Lembre-se: `input()` devolve uma string — você precisa converter para `float` antes de usar.

## 2. Par ou ímpar

Dado um número inteiro, imprima `"par"` ou `"ímpar"`. Dica: use o operador `%`.

## 3. Soma dos múltiplos de 3 ou 5

Calcule a soma de todos os inteiros de `1` a `999` que são múltiplos de `3` ou de `5`. (Este é o primeiro problema do Project Euler — bom problema para fixar `for` + `if`.)

## 4. Reverter uma string

Dada uma string, imprima ela ao contrário, **sem** usar `s[::-1]`. Use um `for` e construa a string reversa caractere a caractere.

## 5. Bisseção generalizada

Adapte a busca por bisseção que vimos em aula para encontrar a raiz **cúbica** aproximada de um número. Compare o número de passos com o que vimos para a raiz quadrada.

## 6. Desafio: maior fator primo

Encontre o maior fator primo de `600851475143`. (Problema 3 do Project Euler. Vai exigir um pouco de pensamento algorítmico — não tente força bruta direto.)
